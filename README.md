# FinPulse - Personal Finance Manager

A full-stack Java application to track personal finances.

## Features
- Track income and expenses
- Set budgets
- Create savings goals
- View financial dashboard

## Tech Stack
- Backend: Spring Boot
- Frontend: React (Vite)
- Database: PostgreSQL

## How to Run
1. Run backend: mvn spring-boot:run
2. Run frontend: npm run dev
